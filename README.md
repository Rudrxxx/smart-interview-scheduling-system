# Smart Interview Scheduling & Evaluation Platform

A full-stack web application to manage campus recruitment drives including applications, interview scheduling, evaluation, and result generation.

## Tech Stack
- Backend: Python (FastAPI/Flask)
- Database: PostgreSQL
- Frontend: React
- Architecture: Layered (Controllers, Services, Repositories)
